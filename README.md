# node guilds

### 기능
1. 유저 관리
  - 회원가입, 로그인, 로그아웃, 유저 탈퇴
2. 길드 관리
  - 길드 검색, 길드 가입, 길드원 보기
3. 레이드 관리
  - 레이드 일정 추가, 레이드 일정 수정, 레이드 일정 보기

### Run Project
```shell
cd client
npm i
npm run build

cd ../server
npm start
```
